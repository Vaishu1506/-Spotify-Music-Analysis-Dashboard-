## 📌 Project Overview

The Spotify Music Analysis Dashboard is built using Microsoft Power BI to explore and visualize Spotify song data. It enables users to analyze music trends, artist performance, album distribution, song popularity, and listening patterns using interactive dashboards.

This project demonstrates advanced Power BI features, including DAX calculations, Power Query transformations, data modeling, and interactive report navigation.
